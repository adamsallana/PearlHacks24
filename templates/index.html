Need help finding notes?

Then this is the website for you. Our website was designed for UNC students to upload their notes for classes they have taken in the past. This way, instead of notes disappearing into thin air, they can be shared with and utilized by future students. Using the search page, select the class you are looking for notes for. All of the classes that have notes in our database will be listed in the dropdown menu. Don't see the class you're looking for? THen you might just have to be the first person to upload notes to our database for that class. Using the upload feature, you can upload pdf notes to be shared with other students. In the future, we hope to implement text recognition software that will allow users to look up keywords.
